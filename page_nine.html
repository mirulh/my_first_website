An unordered list ( <ul> ) is used to create a list of items in no particular order i.e. the order of items is not relevant. By default, the items in this list will be marked with bullets. Whereas, an ordered list ( <ol> ) is used to create a list of items in a specific order.


Ordered list (numbered)

  1. item number 1
  2. item number 2
  3. item number 3
  4. item number 4

Ordered list (list-style-image)

  1. item number 1
  2. item number 2
  3. item number 3
  4. item number 4

Unordered list (bullet point)

  • item number 1
  • item number 2
  • item number 3
  • item number 4

Unordered list (list-style-type:roman)

  • item number 1
  • item number 2
  • item number 3
  • item number 4
