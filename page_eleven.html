Color scheme of hex, and hsl how does it work

Hex follows the RRGGBB value (red, green, blue) and the maximum value of the color is F (9)
HSL stands for Hue-Saturation-Lightness. The maximum value is 360 for red (0 also is a red value)

Example:
HEX: #FF0000 is red (max F on the first 2 value, RR)
HSL: hsl(360, 70%, 50%)

 

Hue: 360
Sat: 70%
Light: 50%

Example:
HEX: #00FF00 is green (max F on the middle 2 value, GG)
HSL: hsl(120, 100%, 30%)

Hue: 120
Sat: 100%
Light: 30%

Example:
HEX: #0000FF is blue (max F on the last 2 value, BB)
HSL: hsl(240, 50%, 80%)

Hue: 240
Sat: 50%
Light: 80%

Color cobination scheme of hex, and rgb how does it work

By using the RRGGBB scheme and altering its value can generate a mixture of color too.
Additionally, adding rgba can set the color's transparency

Example:
HEX: #FFFF00 is yellow (red + green)
RGB: rgb(255, 255, 0)

This is yellow

Example:
HEX: #FF00FF is purple (red + blue)
RGB: rgba(255, 0, 255, .15)

This is purple

Example:
HEX: #00FFFF is cyan (green + blue)
RGB: rgba(0, 255, 255, .25)

This is cyan

Other way is you can use opacity for transparency too:

Example:
HEX: #FFFFFF is white
RGB: rgb(255, 255, 255)
Opacity: 0.25

This is white

Example:
HEX: #000000 is black
RGB: rgb(0, 0, 0)
Opacity: 0.5

This is black
